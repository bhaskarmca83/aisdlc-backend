# aisdlc-backend

Spring Boot 3 / Java 21 sample target application.
This repo is where ImplementationAgent writes code when running the AI SDLC pipeline.

## Run locally
```bash
mvn spring-boot:run
# Starts on http://localhost:8080
```

## Run tests
```bash
mvn test
```

## Stack
- Spring Boot 3.3
- Java 21
- Spring Security + JWT
- Spring Data JPA
- PostgreSQL 16
- JUnit 5 + Testcontainers
